## Environment Configuration

After creating your connection, set up an environment:

1. Navigate to **Settings \> Environments**
2. Click "Add Environment"
3. Provide:
   - Environment name
   - Select your Semantic Layer connection
4. Save the configuration

:::important
Both connection and environment setup must be completed before creating a Semantic Layer project.
:::

## Security Best Practices

- **Service Accounts**: Use dedicated service accounts instead of personal credentials
- **Least Privilege**: Grant minimum required permissions for security
- **Regular Rotation**: Implement credential rotation policies

## Troubleshooting

If you encounter any issues during setup:

### Connection Failed

1. Verify credentials and permissions
2. Check network connectivity
3. Ensure firewall rules allow access

### Authentication Errors

1. Confirm role assignments
2. Verify warehouse permissions
3. Check key pair configuration

### Environment Issues

1. Validate connection settings
2. Confirm warehouse availability
3. Check resource access permissions

<Tip>
  Support
  If issues persist after trying these steps, contact support with specific error messages for faster resolution.
</Tip>
