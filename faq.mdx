title: Frequenty Asked Questions
---

<AccordionGroup>
  <Accordion title="What is 5X and what capabilities does it offer?">
    5X is an all-in-one data platform that integrates data ingestion, warehousing, modeling, orchestration, and business intelligence (BI) capabilities. It provides a comprehensive solution for managing your entire data stack.
  </Accordion>
  <Accordion title="How many data connectors does 5X support?">
    5X supports over 600 pre-built data connectors and can also create custom connectors for specific needs.
  </Accordion>
  <Accordion title="How does 5X handle data warehousing?">
    5X can provision data warehouses like Snowflake for clients who don't have an existing warehouse. It also supports integration with existing warehouses like BigQuery and Snowflake.
  </Accordion>
  <Accordion title="Can 5X integrate with existing data warehouses and BI tools?">
    Yes, 5X can work with your existing data warehouse and integrate with popular BI tools like Tableau or Power BI. It also offers its own BI layer based on Apache Superset.
  </Accordion>
  <Accordion title="What kind of data transformations?">
    5X supports complex data transformations using SQL, dbt and Python scripts.
  </Accordion>
  <Accordion title="How does 5X ensure data security and compliance?">
    5X is SOC 2 Type 2 and on our way to becoming GDPR compliant. It offers field-level hashing for sensitive data, and can implement data masking through SQL queries or custom scripts.
  </Accordion>
  <Accordion title="What is 5X's pricing model?">
    5X uses a consumption-based pricing model with no platform fees or long-term commitments. Costs are based on data volume, compute usage, and number of BI users.
  </Accordion>
  <Accordion title="What is the 48-hour jumpstart program?">
    The 48-hour jumpstart is a free program where 5X builds out a specific use case for a client within 48 hours, demonstrating the platform's capabilities with the client's own data.
  </Accordion>
  <Accordion title="How does 5X handle data residency requirements?">
    5X uses client's warehouse for data storage, allowing clients to choose their data residency location to meet specific regulatory requirements.
  </Accordion>
  <Accordion title="Does 5X offer role-based access control and row-level security?">
    Yes, 5X provides both role-based access control and row-level security options, allowing fine-grained control over data access for different users or customers.
  </Accordion>
  <Accordion title="How does 5X's ingestion process work (initial load vs. incremental updates)?">
    5X performs a full data transfer on the first connection, then uses incremental updates for subsequent syncs, capturing only new or changed data.
  </Accordion>
  <Accordion title="What support and consulting services does 5X offer?">
    5X provides support across multiple time zones and offers consultancy services with a team of data engineers.
  </Accordion>
</AccordionGroup>
